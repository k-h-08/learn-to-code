📖 Lesson

HTML stands for HyperText Markup Language. It tells the browser what each piece of content is on a page.

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello World</h1>
    <p>This is my first HTML page!</p>
  </body>
</html>
